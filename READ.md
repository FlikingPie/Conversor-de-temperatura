# 🌡️ Convertidor de Grados en Python

Aplicación gráfica desarrollada en **Python** utilizando **Tkinter** que permite convertir temperaturas entre diferentes escalas.

## 📌 Características

- Conversión entre:
  - Celsius (°C) → Kelvin (K)
  - Celsius (°C) → Fahrenheit (°F)
  - Kelvin (K) → Celsius (°C)
  - Kelvin (K) → Fahrenheit (°F)
  - Fahrenheit (°F) → Celsius (°C)
  - Fahrenheit (°F) → Kelvin (K)
- Interfaz gráfica sencilla.
- Validación de datos ingresados.
- Mensajes de error cuando el usuario introduce valores inválidos.

## 🖥️ Interfaz

El programa incluye:

- Campo para ingresar los grados.
- Lista desplegable (**Combobox**) para elegir la conversión.
- Campo para mostrar el resultado.
- Mensajes informativos mediante `messagebox`.

## 📂 Tecnologías utilizadas

- Python 3
- Tkinter
- ttk (Combobox)
- messagebox
